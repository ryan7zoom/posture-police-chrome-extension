# Posture Police

A Chrome extension that reminds you to fix your posture randomly every 30–60 minutes and drink water every 2 hours. Keeps your back and hydration in check.

---

## Installation in Chrome

1. Download the project folder to your computer.

2. Open Chrome and go to:
   chrome://extensions

3. Enable **Developer mode** (top right toggle).

4. Click **Load unpacked** and select the project folder.

5. The extension is now active. You will start getting posture and water reminders.

---

## Files

- `manifest.json` – Chrome extension manifest (defines permissions and background service worker).
- `background.js` – Handles notifications and timers for posture and water reminders.
- `icon.png` – Custom extension icon used for notifications and Chrome toolbar.
